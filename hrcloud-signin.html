📍 Geofence Configuration
Set location and radius for each company or site
📡 Location Source Mode
The app uses all available sources automatically: GPS (3–15m, outdoors) → WiFi (10–40m, indoors) → Cell/Network (100–2000m, fallback).
For office geofencing, set radius ≥ 50m to reliably cover WiFi positioning.
💡 Recommended geofence radius by location source: GPS = 30m · WiFi = 80m · Office building = 100–150m · Campus/site = 200–500m

📋 Integration Guide
1   Set up companies and staff in the Setup tab above
2   Configure geofence zones for each office/site
3   Staff sign in daily using their Employee ID
4   Admin exports attendance CSV (HRCloud format)
5   Upload to HRCloud → Import → Attendance tab
⚠ Employee IDs in this Sign-In app must match the Employee IDs in HRCloud exactly for the import to link correctly.
